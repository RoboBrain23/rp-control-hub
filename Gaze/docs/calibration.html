calibration module¶

class calibration.Calibration(calibration_frames=200)[source]¶

Bases: object

Calibration class for calibrating eye threshold and blink threshold

static threshold_calibrate(eye_gaze, thresh)[source]¶

Calibrate eye threshold by comparing number of white pixels in eye region. if percentage of white pixels is less than 20, decrease threshold by 1 if percentage of white pixels is greater than 25, increase threshold by 1

Parameters

  • eye_gaze – Eye object

  • thresh – current threshold

Returns

thresh the updated threshold

blink_calibrate()[source]¶

Calibrate blink threshold by calculating mean of blink ratios that are greater than 4.

None
